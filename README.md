# PRIMI IIT Dockerized YARP + Event-Driven Workstation

This repository provides a Dockerized `YARP` + `event-driven` workstation with:

- one reusable Ubuntu-based image
- GUI tool support on the host desktop
- helper scripts for non-expert users
- `yarpmanager` applications for the same tools and demos

Choose the guide that matches your role:

- [User README](USER_README.md)
- [Developer README](DEVELOPER_README.md)

If you just want to start the workstation and run a demo, go to the [User README](USER_README.md).

If you need to maintain the image, scripts, `yarpmanager` definitions, or the GitHub Actions / GHCR distribution flow, go to the [Developer README](DEVELOPER_README.md).
